Reverse - Unified Color

2021-02-16

The Puzzles of Color. You can Reverse, Rotate and Slide color blocks. There are 18 puzzles. Each have 6 stages with infinite levels. You can play as long as you like.

This APP include some simple and interesting puzzles of color. Each have some modes and levels. All need to think and select.
Unified Color: Click to make all polygons have same color. More polygons will change the color when you click ones. There are three shapes(triangle, square, hexagon) and each have some patterns and six levels.
Rotating Color Puzzle: Click to rotate color blocks to put them in right place. There are three shapes(triangle, square, hexagon) and each have six levels.
Cyclic Color Blocks: Slide the color blocks circularly to make the color same horizontally or vertically and horizontally. Including memory mode which hide the color when you sliding.

This APP have 18 puzzles of color. Each have 6 stages with infinite levels. You can play as long as you like.
